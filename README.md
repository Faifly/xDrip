# xDrip
iOS version of [Nightscout's xDrip+](https://github.com/NightscoutFoundation/xDrip).

## Description


#### Supported transmitters:
- [x] Dexcom G6.

#### Supported platforms:
- [x] iPhone.
- [x] iPad.
- [x] macOS.

#### Other Features:

- [x] Nightscout sync.
- [x] Follower mode.
- [x] Various alerts.
- [x] 69.5% code coverage (aiming for 100%).
- [x] Basal tracking.
- [ ] Transmitter resetting.
